O que é javaScript?

JavaScript é uma linguagem de programação de alto nível e interpretada, utilizada principalmente para criar páginas web interativas e dinâmicas.

É uma das três principais tecnologias web, juntamente com HTML e CSS, e é executada em navegadores web modernos, bem como em servidores e dispositivos móveis.

O JavaScript permite aos desenvolvedores criar e manipular conteúdo da web em tempo real, criar animações e efeitos visuais, validar formulários e interagir com APIs e bancos de dados, entre outras funcionalidades.
